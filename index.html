<Diseña tu web>

¿Cómo se hace una página web? Aquí podrás aprender de forma fácil y rápida a diseñar una página web utilizando el lenguaje de marcado HTML5, a darle estilo con CSS3 e incluso a crear tu primer programa con JavaScript.

#Introducción a los cursos

Si estás buscando un curso para aprender a diseñar páginas web estás en el lugar adecuado. Antes de que empieces con el proceso de aprendizaje debes saber que tanto tu edad como tu nivel de conocimientos no importan. Todo el mundo puede aprender a diseñar una página web desde cero independientemente de su edad y de sus conocimientos. Acepta el desafío de aprender algo nuevo y deja que tu mente y tu mundo se expandan.

Este web site está diseñado de la forma más limpia (en cuanto colores) y clara (en cuanto a fuentes tipográficas) para evitar distracciones y centrarse en lo que realmente nos compete; el código y tu aprendizaje.

Aprender esta nueva disciplina requerirá de mucha práctica. En este curso vamos a hacerlo de una forma pausada y guiada. Comenzaremos con lo más básico y los ejercicios se volverán cada vez más intensos y complejos a medida que avancen las lecciones.

Entenderás como funciona internet y las páginas web, conocerás como se construyen con HTML5 y aprenderás todas las etiquetas y sus propiedades para poder crear su estructura. Te enseñaremos como 'ponerla guapa' aplicándole el estilo que deseas con CSS3 y hacerla interactiva con JavaScript para que así luzca increíble.

El objetivo principal de este curso es que aprendas a utilizar HTML5 y CSS3 para crear una página web ya sea un blog personal, una tienda o cualquier otro negocio que te propongas.

Por último también aprenderás algunas nociones básicas de diseño gráfico para poder utilizar aplicaciones como Photoshop o Indesign con lo que podrías llegar finalmente a convertirte en un diseñador super completo porque además de saber HTML5 y CSS3 podrías llegar a diseñar tu mismo muchos elementos decorativos como los banners, fondos, o ilustraciones de tu site.

Para poder iniciarte en este proceso de aprendizaje necesitarás, obviamente, un ordenador y poco más (además de tu interés y tu paciencia). Digo que poco más ya que, con un simple editor de texto y un navegador podremos escribir el código y lanzarlo para ver su resultado. Nosotros te vamos a recomendar algunas aplicaciones para que la tarea de escribir código sea menos tediosa y mucho más sencilla de llevar a cabo.

Lo primero de todo es que instales en tu ordenador el Visual Studio Code que es una aplicación gratuita muy intuitiva y facil de utilizar con la que te va a resultar muy sencillo aprender a escribir en cualquier lenguaje, tanto de marcas, como de programación, y que incluye soporte para la depuración, control integrado de Git (que ya veremos más adelante), resaltado de sintaxis, finalización inteligente de código, fragmentos y refactorización de código.

Como navegador de internet elegiremos el Google Chrome o el Firefox de Mozilla (el que más te guste). Ya que consideramos que son los dos mejores porque ofrecen la posibilidad de inspeccionar elemento a elemento del código y hacer cambios on the fly de mejor manera que el resto. Si puedes evitar el Internet Explorer mejor que mejor.

En el mercado existen multitud de sistemas de gestión de contenidos como Wordpress, Joomla, Drupal, etc que te facilitan la incorporación temas, plantillas, widgets, plugins, etc para la realización de una página web pero como vamos a enseñarte a crear una web desde cero vamos a obviarlos totalmente.

Antes de ponerte a crear tu página web hay una serie de puntos a tener en cuenta:

1. Define cuál será el objetivo de la página web y enumera los requisitos para cumplirlos

Lo primero es lo primero: ¿de qué se tratará tu sitio web? Puede ser de un negocio ya establecido o de alguno nuevo que tengas en mente. Lo importante es tener claro qué quieres conseguir con esta página web.

¿Para qué quieres crear una página web? Puede ser para vender, promocionar u ofrecer tus productos o servicios, mostrar tu portafolio, captar nuevos consumidores, dar visibilidad a tu marca, crear una comunidad, hacer contenido educativo, tener un blog, entre otros.

Saber por qué quieres crear un sitio web te permitirá determinar:

Qué páginas quieres incluir en tu estructura. Algunos ejemplos de elementos fundamentales en tu web: inicio (home), sobre el negocio, quién es el equipo, página de precios, portafolio, de contacto, proyectos, servicios.

Los tipos de contenidos que te gustaría incluir. Esta es la primera incursión y, por tanto, debes dejar volar tu imaginación. Piensa en grande y explora tu creatividad. Puede ser una galería de imágenes, artículos de blog, vídeos, logos, testimonios de clientes, servicios y su descripción, archivos descargables.

Qué otras herramientas puedes necesitar y cómo conectarlas con tu sitio web. Si quieres quieres tener una plataforma para enviar campañas de email marketing o chat online, aprovecha de conectar estas herramientas a tu sitio. A lo mejor no tienes todo claro sobre este punto y, por eso, es fundamental que te hagas la pregunta: ¿Necesito un sitio web?

Si lo primero que vino a tu mente fue “sí”, entonces tómate el tiempo necesario para descubrir el “para qué”.

Una vez que lo hayas comprendido, avanza hacia el siguiente paso en el que investigas el mercado y a tu competencia.

2. Investiga el mercado del negocio y la competencia

Con tu objetivo claro y las primeras ideas en mente, es momento de investigar el mercado y analizar la competencia.

- ¿Qué investigar del mercado y la competencia al crear una página web desde cero?

- Qué le interesa a los potenciales clientes de tu negocio.

- Qué contenido está publicando la competencia que atrae a esos potenciales clientes de tu negocio.

La mejor manera de realizar esta tarea es bucear en distintos espacios donde están tus leads.

Por ejemplo, revisar los canales de comunicación de tus competidores, páginas web, foros, búsquedas de Google, y redes sociales.

Fíjate en todo lo que puedas. Toma nota mental de los temas que hablan, los formatos de contenido que utilizan para comunicar, el estilo que ocupan al hablar y cómo presentan la información.

Para que te hagas una idea piensa que tu eres un entrenador personal. Mirando las páginas web de otros entrenadores personales puedes notar que todos tienen una estructura de similar:

Página de servicios donde describen cada uno de los servicios que ofrecen y sus planes de precios.

“Sobre mí” donde el entrenador personal cuenta su experiencia.

Clientes felices que incluyen testimonios y transformaciones.

Blog con distintos artículos que ayudan a posicionar la página web a través de distintos objetivos: educar, dar tips, informar, contar historias, entre otras cosas.

Formulario para suscribirse al newsletter y a cambio descargar un plan de recetas.

Link directo a las redes sociales donde privilegian el perfil de YouTube e Instagram, lo cual te da un indicio de qué redes sociales son más populares entre los entrenadores personales.

Además de los puntos descritos anteriormente, imagínate que revisando grupos de deportistas en redes sociales, te das cuenta que continuamente hacen preguntas sobre centros de entrenamiento.

Eso es un excelente indicador de que tu página web podría tener una lista de los gimnasios o lugares de entrenamiento donde como entrenador personal prestas servicios.

La investigación es clave. Ya sea si estás buscando cómo crear una página web desde cero para un negocio que tenga un retorno alto o para una obra de caridad meramente informativa.

Con una correcta búsqueda vas a poder determinar a cabalidad qué quieren ver tus potenciales clientes o consumidores y planear tu contenido de acuerdo a ello.

Ahora que ya tienes claro qué incluir en tu sitio web, es momento de avanzar al tercer paso a tantear qué te atrae visualmente.

3. Explora opciones de sitios web que sean atractivos visualmente

En el paso anterior, concentramos los esfuerzos en investigar qué contenido incluir en la página web. En este punto también habrá que investigar y buscar información, pero netamente visual.

Aquí deberás explorar las diferentes plantillas y analizar los sitios web de tu competencia, ya que así sabrás qué va mejor con tu audiencia.

¿En qué fijarse al mirar otras páginas web?

Al igual que en el punto anterior, analiza cómo son visualmente las páginas web de tus competidores.

Presta atención a:

Colores que usan, tipografías, imágenes (tamaño, cuántas, en qué secciones), formas geométricas cuadradas o redondas, botones con CTA o llamados a la acción.

Además, inspírate revisando diferentes sitios web o mirando Pinterest. Una buena manera de hacer la lluvia de ideas para hacer la parte visual de tu página web es crear un tablero en Pinterest donde puedas ir guardando ideas, colores, formas, tipografías, o estilos de imágenes que te interesen.

Al crear la parte visual de una página desde cero ten en cuenta estos tres puntos:

El objetivo de tu página web

Cuando estés en esta etapa, no pierdas de vista el objetivo de tu web. Si lo que buscas con tu sitio es generar más conversiones, tu página deberá tener más llamados a la acción y tendrás que elegirlos muy bien.

O si tu web es un portafolio de tus proyectos y buscas promocionarte, procura tener imágenes de alta calidad y una breve explicación de cada uno de los trabajos que has hecho.

La audiencia a la que te diriges

Ten en cuenta tu audiencia, a quién va dirigida la página web. En ese sentido, si está dedicada a personas mayores, intenta que la tipografía sea fácil de leer, tenga un tamaño adecuado y un buen contraste de la letra con el color o la imagen de fondo.

Y, por ejemplo, si el sitio está orientado a jóvenes, que sea con colores llamativos, tenga una letra juvenil, entre otras cosas.

El tamaño de tu negocio

Si vas a mirar sitios web de la competencia, garantiza que sean de compañías o negocios del mismo tamaño que el tuyo. No pierdas tiempo mirando páginas web de compañías mucho más grandes que la tuya.

Sé realista, ya que lo más probable es que el presupuesto que tengan sea más elevado y los objetivos del sitio también sean distintos.

Tener presente los tres puntos descritos anteriormente, te va a ayudar a decidir cómo será visualmente tu página web para que la construyas de cero.

Además, te permitirá aclarar si necesitas una página simple o one-page o algo más complejo como una multi-page.

Ya pasamos lo más difícil de crear una página web desde cero. Con objetivos claros, con una correcta investigación de la competencia y con una exploración visual realizada, toca pasar a la parte técnica de la construcción de una web.

4. Selecciona cuál será el alojamiento de tu página web

Al construir una página web desde cero deberás decidir dónde alojarla, esto quiere decir, dónde estará almacenada la página web.

¿Por qué es importante elegir el hosting adecuado? Veamos los siguientes factores que lo demuestran:

Perfomance de tu sitio web

El alojamiento web o hosting puede afectar la velocidad de carga y performance que tenga tu sitio web. Por eso, es clave que compares la calidad de cada uno de los hosting que estés evaluando.

Escalabilidad y seguridad

La escalabilidad de tu sitio web y seguridad podrían verse amenazadas si no eliges el hosting adecuado, así que no ignores esos puntos.

Respaldo de archivos

Cuando crees tu sitio web es mejor tener todas las variables bajo control. Y si por algún motivo necesitas recuperar todo el contenido, vale la pena contar con un hosting que tenga respaldo de todo.

Soporte técnico

uede que durante la construcción o después te enfrentes a alguna duda que un especialista necesitará resolver. Si esto sucede, te conviene tener un hosting que tenga el soporte técnico incorporado.

Como vimos en los puntos anteriores, la elección de hosting no puede ser al azar. Esta, incluso, puede ser determinante para la efectividad de tu sitio web. Sin embargo, la buena noticia es que la mayoría de los hosting populares cuentan con capacidades robustas.

Y las diferencias entre ellas solo las verán compañías grandes que atraen miles de millones de visitas cada mes. Si tu página web es para tu negocio aún pequeño, no te será difícil encontrar el hosting adecuado.

Antes de decidirte por un hosting en concreto, te recomiendo que investigues en sus respectivas web y busques al menos que cumplan (en resumen) estos requisitos:

  • Facilidad y rapidez para actualizar contenido.
  • Planes de hosting diferentes para poder migrar en caso de necesidad. Suele haber diferencias en: almacenamiento, transferencia de datos y número de dominios disponibles.
  • Correo electrónico ilimitado (número de cuentas de email con tu dominio.
  • Filtros de spam (imprescindible).
  • Interfaz de uso fácil e intuitivo (cpanel).
  • Soporte técnico de calidad, a ser posible 24 h/7 días semana/365 días al año.
  • Que tenga foros, apartado de Preguntas Frecuentes (FAQ), vídeo tutoriales, etc. Muchas de nuestras dudas se podrán resolver haciendo una búsqueda en la web del hosting.
  • Copias de seguridad autómaticas y programables.

5. Ponte creativo y elige cuál será el nombre del dominio

Llegó el momento de elegir el nombre de dominio que representará tu negocio. ¿A qué nos referimos con dominio? Es la dirección que tendrá tu sitio o también conocido como URL. Esta es la manera en que tu audiencia te reconocerá o buscará al momento de visitar tu sitio web.

Un nombre de dominio es una cadena de texto que se asigna a una dirección IP numérica, que se utiliza para acceder a un sitio web desde el software cliente. Dicho de forma más sencilla, un nombre de dominio es el texto que un usuario escribe en una ventana del navegador para llegar a un sitio web concreto. Por ejemplo, el nombre de dominio de Google es "google.com".

Todos los nombres de dominio son gestionados por los registros de dominios, que delegan la reserva de los mismos en los registradores. El que quiera crear un sitio web puede registrar un nombre de dominio en un registrador y, en la actualidad, hay más de 300 millones de nombres de dominio registrados.
